# 💸 Monthly Expense Tracker for AUIers

A simple and interactive web app to track monthly expenses by category, built with vanilla HTML, CSS, and JavaScript.

---

## ✨ Features

- Track expenses across **6 categories**: Housing, Food, Transportation, Academics, Leisure, and Others
- **Doughnut chart** visualization powered by Chart.js
- **Month & year selector** to navigate between periods
- **Persistent data** — expenses are saved in the browser via localStorage
- Fully **responsive** — works on mobile and desktop

---

## 🖥️ Demo

Open `index.html` directly in your browser — no server or installation needed.
